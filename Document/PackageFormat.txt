
Package Format for Read/Write IIC Device via USB

-----------------------------------------------------------
| Offset | Length | Description
+--------+--------+----------------------------------------
|  0x00  |    3   | Package head, always 0xAAAAAA
+--------+--------+----------------------------------------
|  0x03  |    1   | Command, Write or Read, see usb2iic.h
+--------+--------+----------------------------------------
|  0x04  |    1   | IIC Device Address
+--------+--------+----------------------------------------
|  0x05  |    2   | Memory Address, 2 bytes
+--------+--------+----------------------------------------
|  0x07  |    1   | Read or Write Length
+--------+--------+----------------------------------------
|  0x08  |        | 
|  ~~~~  |   32   | N bytes data
|  0x28  |        | 
+--------+--------+----------------------------------------
|  0x3C  |    3   | Package Tail, alwayx 0xF0F0F0
-----------------------------------------------------------
